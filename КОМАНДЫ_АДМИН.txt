========================================
КОМАНДЫ ДЛЯ ВЫПОЛНЕНИЯ В POWERSHELL ОТ АДМИНИСТРАТОРА
========================================

ВАЖНО: Запустите PowerShell от имени администратора!

1. Переход в папку проекта:
   cd E:\ORCHID

2. Инициализация Git:
   git init

3. Добавление всех файлов:
   git add .

4. Создание коммита:
   git commit -m "Initial commit: ORCHID project with admin panel"

5. Настройка ветки:
   git branch -M main

6. Подключение к GitHub:
   git remote add origin https://github.com/andreybatraider/orchid.git

7. Пуш на GitHub:
   git push -u origin main

========================================
КАК ЗАПУСТИТЬ POWERSHELL ОТ АДМИНИСТРАТОРА:
========================================

1. Нажмите Win + X
2. Выберите "Windows PowerShell (Администратор)"
   или "Терминал (Администратор)"
3. Подтвердите запрос UAC
4. Выполните команды выше
