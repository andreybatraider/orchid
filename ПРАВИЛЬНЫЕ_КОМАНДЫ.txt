ПРАВИЛЬНЫЕ КОМАНДЫ ДЛЯ ПУША НА GITHUB
ВАЖНО: Выполняйте в директории E:\ORCHID
========================================

1. ПЕРЕХОД В ПАПКУ ПРОЕКТА:
   cd E:\ORCHID

2. Инициализация Git:
   git init

3. Добавление всех файлов:
   git add .

4. Создание первого коммита:
   git commit -m "Initial commit: ORCHID project with admin panel"

5. Настройка ветки main:
   git branch -M main

6. Подключение к GitHub:
   git remote add origin https://github.com/andreybatraider/orchid.git

7. Пуш на GitHub:
   git push -u origin main
